# 1924 — PROVENANCE: Le Flagship Numérique

> "Le vrai luxe ne réclame pas d'attention ; il attend dans le silence de nos archives."

---

## I. Le Concept (L'Exclusivité)
A **PROVENANCE** redefine a interação digital ao afastar-se do modelo de "e-commerce" de massa. Esta plataforma funciona como um **Concierge de Haute Lignée**. O objetivo não é a transação, mas a **alocação**. O design foi concebido para que cada clique pareça um convite e cada página um capítulo de um livro de arte.

A mensagem é clara: *L'Élite, pas la Masse.*

---

## II. Architecture Visuelle : Quiet Luxury
O design adota o rigor estético das casas de alta-costura da *Place Vendôme*:

* **Obsidian & Ivoire :** Uma paleta que foge do branco puro, utilizando tons de marfim e preto profundo para evocar documentos históricos e perenidade.
* **Espace Négatif :** O uso de margens amplas serve para dar "respiro" às peças, tratando cada item da API como um artefato raro em uma galeria privada.
* **Typographie de Transition :** O contraste entre a autoridade da *Cormorant* e a fluidez heráldica da *Cormorant Garamond Italic*.

---

## III. Les Sections (La Nomenclature)
Ao contrário de plataformas convencionais, a navegação segue o protocolo das grandes *Maisons*:

| Concept | Nomenclature | Description |
| :--- | :--- | :--- |
| **Home** | `La Maison` | O epicentro da herança e história da linhagem. |
| **Inventory** | `Le Cellier` | Onde os exemplares são conservados sob guarda rigorosa. |
| **Selection** | `Le Chef-d'Œuvre` | Análise profunda da procedência de uma peça única. |
| **Registry** | `L'Atelier` | Onde o protocolo de intenção de acesso é formalizado. |

## IV. Fonctionnalités de Haute Spécification

### A. Le Système de Privilège
A plataforma reconhece a distinção do usuário através de algoritmos de mérito. Aplicamos o **Privilège de Lignée (—15%)** de forma visualmente solene, utilizando o tom *Rouge Bordeaux* para sinalizar que o valor exibido é uma deferência exclusiva ao membro.

### B. Protocole de Conversion (EUR €)
Para manter a etiqueta europeia, todos os dados são normalizados para **Euro (€)** no momento da ingestão. Isso isola a experiência da volatilidade externa, mantendo a estabilidade visual de uma moeda de prestígio e tradição.

### C. Registre de Persistance
As escolhas do usuário não são meros "itens de consumo", mas uma **Intention d'Acquisition**. Estes dados são preservados via `localStorage`, funcionando como um registro histórico e persistente das aspirações do membro dentro da plataforma.

---

## V. Spécifications Techniques
* **Engine:** React.js com arquitetura de componentes de alta fidelidade.
* **Navigation:** Navbar adaptativa com estados `at-top` e `nav-hidden` para imersão total.
* **Data:** Integração assíncrona com normalização de preços na camada de serviço.
